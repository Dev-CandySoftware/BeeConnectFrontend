# Ghid de utilizare - BeeConnect

Acest document explică pe scurt cum folosești aplicația BeeConnect în fluxul de zi cu zi.

## 1. Navigare în aplicație

După ce pornești aplicația, folosești meniul lateral (sidebar):

- `Acasă` - ecran principal;
- `Contacte` - listă și management contacte;
- `Grafice` - statistici;
- `Ședințe` - listă și management ședințe;
- `Cursuri` - secțiune dedicată cursurilor.

## 2. Gestionare contacte

### Adăugare contact

1. Mergi la `Contacte`.
2. Apasă `Adaugă contact`.
3. Completează formularul:
   - nume, prenume, email, telefon;
   - câmpuri suplimentare (sursă, venit, vârstă etc.);
   - observații și data contractului.
4. Apasă `Salvează`.

### Editare contact

1. Mergi la `Contacte`.
2. Pe cardul contactului dorit, apasă `Editează`.
3. Modifică datele.
4. Apasă `Salvează`.

### Ștergere contact

1. Mergi la `Contacte`.
2. Pe cardul contactului dorit, apasă `Șterge`.
3. Contactul este eliminat din listă.

## 3. Gestionare ședințe

### Listă ședințe

În `Ședințe` vezi:

- consultantul;
- ora de început și ora de sfârșit;
- observațiile;
- data creării.

### Adăugare ședință

1. Mergi la `Ședințe`.
2. Apasă `Adaugă ședință`.
3. Completează formularul:
   - nume consultant;
   - început;
   - sfârșit;
   - observații.
4. Apasă `Salvează`.

### Editare ședință

1. Mergi la `Ședințe`.
2. Apasă `Editează` pe cardul dorit.
3. Modifică intervalul sau alte date.
4. Apasă `Salvează`.

### Ștergere ședință

1. Mergi la `Ședințe`.
2. Apasă `Șterge` pe cardul dorit.
3. Ședința este eliminată din listă.

## 4. Dashboard și grafice

În `Grafice` poți vedea distribuții pentru:

- venit;
- vârstă;
- ocupație;
- stare civilă;
- număr copii;
- frecvență întâlniri;
- și alte câmpuri relevante.

Paleta de culori este adaptivă în funcție de dimensiunea ecranului.

## 5. Erori frecvente

- **Nu se încarcă datele**: verifică dacă backend-ul este pornit.
- **Nu se salvează formularul**: verifică validările obligatorii (ex: câmpuri goale, interval invalid).
- **Ora pare greșită la ședințe**: verifică timezone-ul backend-ului și formatul `DateTime`.

## 6. Recomandări de utilizare

- păstrează datele de contact complete pentru statistici mai utile;
- verifică intervalele ședințelor înainte de salvare;
- rulează periodic dashboard-ul pentru a observa distribuțiile importante.
